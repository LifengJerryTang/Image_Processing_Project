# Image Processing API

## Getting Started
### Installing all the dependencies:
```
npm install
```
### Building the project to production:
```
npm run build
```
### Running the server:
```
npm run start
```
### Run all the tests:
```
npm run test
```

### Run lint:
```
npm run lint
```

## Endpoint(s)
### Creating and getting the thumbnail of an image:
```
GET /api/images/?filename={IMAGE_NAME}&height={HEIGHT}&width={WIDTH}
```
